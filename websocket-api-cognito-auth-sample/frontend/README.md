# Frontend for demo
A frontend app only for demo purpose, with React, Amplify libraries, MUI, and Vite.

![ui](../doc/img/ui.png)

## Run locally
Create `.env.local` first.

```sh
cp .env.sample .env.local
# fill values according to the outputs of the CDK stack you deployed to AWS.
```

Then run Vite.

```sh
npm run dev
```
